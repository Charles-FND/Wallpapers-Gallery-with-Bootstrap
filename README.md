# Premium Image Gallery

This is a simple, responsive image gallery web page created using HTML, CSS, JavaScript, and Bootstrap.

## Features

*   Modern navbar with a stylish title
*   Gradient 'Add Image' button with hover effects
*   Clean, light gradient background
*   File upload dialog to select images
*   Uploaded images are automatically added to a gallery
*   Bootstrap grid layout for responsiveness
*   Rounded corners, shadows, and hover zoom animations on gallery items
*   Fade-in transitions for new images

## How to Run

1.  Clone this repository.
2.  Open the `index.html` file in your web browser.

That's it! No special build steps are required.
